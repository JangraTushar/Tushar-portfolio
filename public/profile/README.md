# Profile Picture Directory

This directory is created specifically for storing your portfolio profile picture.

## How to use:
1. Save your own profile photo in this folder.
2. The recommended file name is `profile.jpg`, `profile.png` or `avatar.svg`.
3. If you name it anything else, you can adjust the filename in the standard data configs or let the portfolio's fallback system automatically resolve it.

Currently, the portfolio references:
- Primary path: `/profile/profile.jpg` (or `/profile/profile.png`)
- Fallback path: `/profile/avatar.svg` (default high-end vector avatar)
- Smart online fallback: Industry standard developer placeholder image from a secure unsplash CDN.
